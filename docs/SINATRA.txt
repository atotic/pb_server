- Middleware is the simplest thing in the world:

class ResponseTimer
  def initialize(app)  
    @app = app  
  end  
    
  def call(env)  
    status, headers, response = @app.call(env)  # you always call the app that came in the initializer
    [status, headers, "<!-- Response Time: ... -->\n" + response.body]  
  end  
end  

To configure middleware in Sinatra, call
use(middleware, *args, &block)
